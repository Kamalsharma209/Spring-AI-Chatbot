# Spring AI Chatbot

A chatbot built using Spring Boot and Spring AI with Ollama.

## Tech Stack

- Java 21
- Spring Boot 3.5
- Spring AI
- Ollama
- Maven

## Features

- AI-powered chatbot
- REST API
- Ollama integration
- Layered architecture (Controller, Service, DTO)
